# Task-05: E-Commerce Checkout Automation

## 📌 Internship Project – Prodigy InfoTech

### 🔹 Objective
To automate the complete checkout workflow of an e-commerce web application using Selenium.

---

## 🛠 Tools & Technologies Used
- Python
- Selenium WebDriver
- Google Chrome Browser

---

## 🌐 Test Website
https://www.saucedemo.com/

---

## ✅ Test Scenario Covered
1. Launch browser
2. Login with valid credentials
3. Verify successful login
4. Add product to cart
5. Navigate to cart page
6. Proceed to checkout
7. Fill customer details
8. Complete the purchase
9. Validate order confirmation message

---

## 🎯 Expected Result
User should successfully complete the checkout process and see the confirmation message:

"Thank you for your order!"

---

## 📌 Actual Result
Test executed successfully and checkout process completed without errors.
